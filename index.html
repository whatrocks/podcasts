Top Podcasts among Farcasters

Add your podcasts to this page:

  1. 1. Export your OPML XML file from your podcast player
  2. 2. Create a public Github gist with the contents of this XML file
  3. 3. Reply to this cast on Farcaster with the "raw" link from GitHub Gist

Note: If you are using Apple Podcasts, it is not possible to grab your OPML file anymore.

Instead, you can use the plist file on your Mac if you have the Podcasts App on your Mac.

Run this command in your Terminal 

pbcopy < ~/Library/Containers/com.apple.podcasts/Data/Documents/PodcastsDB.plist
and create a Gist with this XML file in your clipboard. Then follow step #3 above!
